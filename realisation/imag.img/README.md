#  Blog Solicode - Base de données

## Description
Ce projet consiste à créer une base de données relationnelle SQL pour un blog.

## Objectifs
- Concevoir une base de données structurée
- Utiliser les clés primaires et étrangères
- Assurer l’intégrité des données

## Tables
- users
- categories
- posts
- comments

## Relations
- Un utilisateur peut créer plusieurs articles
- Un article appartient à une catégorie
- Un article peut avoir plusieurs commentaires

##  Technologies
- MySQL
- PhpMyAdmin
- SQL

## Installation
1. Importer le fichier `script.sql` dans PhpMyAdmin
2. La base `blog_solicode` sera créée automatiquement
